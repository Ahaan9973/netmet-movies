## 🛠 Setup

### 1. Get a Free TMDB API Key
- Sign up at [developer.themoviedb.org](https://developer.themoviedb.org)
- Go to Settings → API → Copy your key

### 2. Add Your API Key
Open `index.html` and find this line:
```js
const TMDB_KEY = 'YOUR_TMDB_API_KEY_HERE';
```
Replace with your actual key.

### 3. Deploy to GitHub Pages
1. Upload `index.html` to this repository
2. Go to **Settings → Pages**
3. Set branch to `main` → Save
4. Your site goes live at `https://YOUR-USERNAME.github.io/netmet-movies`
